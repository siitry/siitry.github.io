发表于 2019-08-26 更新于 2020-03-25 Valine：

hexo 作为静态博客很受欢迎,除了可以应该用github上大量主题之外
还可以开启友链、打赏、评论等互动功能

hexo 搭载了众多可使用的评论系统,如：

畅言,多说,有言,disqus…等等
目前本博客使用 valine 评论系统
原因有三：
1.轻量级
2.注册快捷
3.简单优雅大方 markdown语法支持
4.官方解释：Leancloud的快速、简洁且高效的无后端评论系统
